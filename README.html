nlpsandboxclient¶

nlpsandboxclient is a set of tools and commands that provides an interface for NLP sandbox data node, annotation services, and evaluation of annotation results.

nlpsandboxclient is written in Python and uses the synapseclient library to pull information from Synapse (account required).

Installation¶

To install from pypi:

$ pip install nlp-sandbox-client

To install from the source:

$ python setup.py install
